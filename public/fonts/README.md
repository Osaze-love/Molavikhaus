# Circular Font Files

Place your Circular font files in this directory. The font loader expects the following files:

- `Circular-Book.woff2` (weight: 400, normal)
- `Circular-BookItalic.woff2` (weight: 400, italic)
- `Circular-Medium.woff2` (weight: 500, normal)
- `Circular-MediumItalic.woff2` (weight: 500, italic)
- `Circular-Bold.woff2` (weight: 700, normal)
- `Circular-BoldItalic.woff2` (weight: 700, italic)

## Getting Circular Font

Circular is a commercial font by Lineto. You can:

1. Purchase it from [Lineto](https://lineto.com/)
2. Or use a similar open-source alternative like Inter or DM Sans

If you have the font files in a different format (TTF, OTF), you can convert them to WOFF2 using online tools.
